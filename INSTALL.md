# Installation

## Dependencies

The following are required:

* python (>=3.5)
* python-qt4
* Mujoco 1.50

```sh
python3 -m pip install --user --upgrade -r requirements.txt
```

For additional OPC UA support:

```sh
python3 -m pip install --user --upgrade -r requirements-opcua.txt
```


## Installation
```sh
sh compile_proto.sh
```
