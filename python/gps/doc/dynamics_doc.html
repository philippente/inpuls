Code

The fitting function takes as arguments:

  • X: States of the trajectory samples from the previous policy
  • U: Actions of the trajectory samples from the previous policy
def fit(self, X, U):

We get the number of samples, the number of timesteps and the dimensions of the states and the actions from the policy object:

N, T, dimX = X.shape
dimU = U.shape[2]

We use slice syntax so that sigma[index_xu, index_x] means Σxu,x′{\bold \Sigma}_{{\bold x}{\bold u},{\bold x}'}Σ​xu,x​′​​​​ etc.

index_xu = slice(dimX + dimU)
index_x = slice(dimX + dimU, dimX + dimU + dimX)

We obtain the regularization term for the covariance:

sig_reg = np.zeros((dimX + dimU + dimX, dimX + dimU + dimX))
sig_reg[index_xu, index_xu] = self._hyperparams['regularization']

We compute the weight vector and matrix, used to compute sample mean and sample covariance:

dwts = (1.0 / N) * np.ones(N)
D = np.diag((1.0 / (N - 1)) * np.ones(N))

We allocate space for F{\bold F}F, f{\bold f}f and Σdyn{\bold \Sigma}_{dyn}Σ​dyn​​:

self.Fm = np.zeros([T, dimX, dimX + dimU])
self.fv = np.zeros([T, dimX]) 
self.dyn_covar = np.zeros([T, dimX, dimX])

We iterate over ttt and assemble

ytn=(xtnutnxt+1n) {\\bold y}\_t^n=\\begin{pmatrix}{\\bold x}\_t^n\\\\{\\bold u}\_t^n\\\\{\\bold x}\_{t+1}^n\\end{pmatrix} y​t​n​​=​⎝​⎛​​​x​t​n​​​u​t​n​​​x​t+1​n​​​​​⎠​⎞​​

where the superscript nnn denotes the number of the sample.

for t in range(T - 1): 
    Ys = np.c_[X[:, t, :], U[:, t, :], X[:, t + 1, :]]

We obtain the hyperparameters of the normal-inverse-Wishart prior NIW(μ0,Φ,m,n0)NIW(\mu_0,\Phi,m,n_0)NIW(μ​0​​,Φ,m,n​0​​)

mu0, Phi, mm, n0 = self.prior.eval(dimX, dimU, Ys)

We compute the empirical mean and empirical covariance

μemp,t=1N∑n=1Nytn {\\bold \\mu}\_{emp,t}=\\frac{1}{N}\\sum\_{n=1}^N{\\bold y}\_t^n μ​emp,t​​=​N​​1​​​n=1​∑​N​​y​t​n​​
Σemp,t=1N−1∑n=1N(ytn−μemp,t)(ytn−μemp,t)T {\\bold \\Sigma}\_{emp,t}=\\frac{1}{N-1}\\sum\_{n=1}^N({\\bold y}\_t^n-{\\bold \\mu}\_{emp,t})({\\bold y}\_t^n-{\\bold \\mu}\_{emp,t})^T Σ​emp,t​​=​N−1​​1​​​n=1​∑​N​​(y​t​n​​−μ​emp,t​​)(y​t​n​​−μ​emp,t​​)​T​​
empmu = np.sum((Ys.T * dwts).T, axis=0)
diff = Ys - empmu
empsig = diff.T.dot(D).dot(diff)                                       
empsig = 0.5 * (empsig + empsig.T)

We use the empirical mean as our estimated mean and use the normal-inverse-Wishart posterior to get the estimate for the covariance:

μt=μemp,t {\\bold \\mu}\_t={\\bold \\mu}\_{emp,t} μ​t​​=μ​emp,t​​
Σt=Φ+(N−1)Σemp,t+NmN+m(μemp,t−μ0)(μemp,t−μ0)TN+n0 {\\bold \\Sigma}\_t=\\frac{{\\bold \\Phi}+(N-1){\\bold \\Sigma}\_{emp,t}+\\frac{Nm}{N+m}({\\bold \\mu}\_{emp,t}-{\\bold \\mu}\_0)({\\bold \\mu}\_{emp,t}-{\\bold \\mu}\_0)^T}{N+n\_0} Σ​t​​=​N+n​0​​​​Φ+(N−1)Σ​emp,t​​+​N+m​​Nm​​(μ​emp,t​​−μ​0​​)(μ​emp,t​​−μ​0​​)​T​​​​
mu = empmu 
sigma = (Phi + (N - 1) * empsig + (N * mm) / (N + mm) *
    np.outer(empmu - mu0, empmu - mu0)) / (N + n0) 
sigma = 0.5 * (sigma + sigma.T)
sigma += sig_reg

Σt{\bold \Sigma}_tΣ​t​​ can contain singularities so that its inverse contains infinities. To prevent that we add a small regularization term.

Now we condition the gaussian on xxx and uuu:

F=Σx′,xuΣxu,xu−1=(Σxu,xu−1Σxu,x′)T {\\bold F} = {\\bold \\Sigma}\_{x',xu}{\\bold \\Sigma}\_{xu,xu}^{-1} = ({\\bold \\Sigma}\_{xu,xu}^{-1}{\\bold \\Sigma}\_{xu,x'})^T F=Σ​x​′​​,xu​​Σ​xu,xu​−1​​=(Σ​xu,xu​−1​​Σ​xu,x​′​​​​)​T​​
f=μx−Fμxu {\\bold f} = {\\bold \\mu}\_x-{\\bold F}{\\bold \\mu}\_{xu} f=μ​x​​−Fμ​xu​​
Σdyn=Σx′,x′−FΣxu,xuFT {\\bold \\Sigma}\_{dyn} = {\\bold \\Sigma}\_{x',x'}-{\\bold F}{\\bold \\Sigma}\_{xu,xu}{\\bold F}^T Σ​dyn​​=Σ​x​′​​,x​′​​​​−FΣ​xu,xu​​F​T​​
Fm = np.linalg.solve(sigma[index_xu, index_xu],
                     sigma[index_xu, index_x]).T
fv = mu[index_x] - Fm.dot(mu[index_xu])
dyn_covar = sigma[index_x, index_x] - Fm.dot(sigma[index_xu, index_xu]).dot(Fm.T)
dyn_covar = 0.5 * (dyn_covar + dyn_covar.T)

We store F{\bold F}F, f{\bold f}f and Σdyn{\bold \Sigma}_{dyn}Σ​dyn​​:

self.Fm[t, :, :] = Fm
self.fv[t, :] = fv
self.dyn_covar[t, :, :] = dyn_covar

After that, the loop ends and we return F{\bold F}F, f{\bold f}f and Σdyn{\bold \Sigma}_{dyn}Σ​dyn​​:

return self.Fm, self.fv, self.dyn_covar
